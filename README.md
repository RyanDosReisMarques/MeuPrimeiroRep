# Guia do github
## 1° passo
 Dentro do github clique no botão "code"
## 2° passo
 Vai aparecer um link =,copie e coloque no navegador
## 3° passo
 Entre no explorador de arquivos e vai em este computador e aperte em "Usesr U:", busque a pasta com seu nome
## 4° passo
 Crie uma pasta chamada Git
## 5° passo 
 Entre na pasta e aperte o botão direito do mouse e va em "Abrir terminal"
## 6° passo
 Dentro do terminal digite o codigo "git clone "e insira o link copiado ( https://github.com/RyanDosReisMarques/MeuPrimeiroRep.git)
## 7° passo
 Logo após isso você estará dentro da pasta, use "cd" para entrar na pasta "MeuPrimeiroRep", "cd MeuPrimeiroRep"
## 8° passo 
 Após isso coloque o codig "code ." para entrar no VS code, no diretorio que estavamos, "MeuPrimeiroRep" direto no README que marcamos no momento que estavamos fazendo o repositorio no Github
## 9° passo
 Escrevemos "#nome (Ryan)" e alteramos o repositorio, por conta disso temos que salvar
## 10° passo
 Voltamos ao CMD para digitar "git add . ", ele cria uma fila de alterações que fizemos para enviar com o commit para o Git hub
## 11° passo
 Depois use o codigo " git commit -m " (é um comentario que fizemos, que no caso foi acrescentar "Adicionar seu nome")
## 12° passo
 Logo após isso descreva o codigo " git push -u origin main " para poder enviar as alterações que fixamos no main do projeto
